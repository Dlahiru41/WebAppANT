This is a simple web application with a back end database.

The application was built using Netbeans with JavaDB and Glassfish.


The implementation of this application demonstrates the following:-
  1. Setting a connection in oder to connect to a database on Glassfish server
  2. Generating Entity Classes from database tables
  3. Generating JSF pages from Entity classes which allow database CRUD operations
  4. An implementation of database Master-Details views
  5. Java code to connect to a database and execute a SQL query
  6. A AJAX search box to search e.g. a database table or a ResultSet from the execution of a SQL query
  7. Simple Session tracking
  8. Simple User input validation with AJAX
  9. Using Facelets Template and Facelets Template Client in order to create a consistent look and feel of a website
